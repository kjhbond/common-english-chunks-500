영어 5형식 예문 TTS 공부

5형식은 주어 + 동사 + 목적어 + 목적격보어입니다. 핵심은 목적격보어가 목적어를 설명한다는 점입니다.

각 예문은 원어민식 의미 덩어리로 끊어져 있고, Play를 누르면 en-US TTS가 덩어리별로 짧게 멈추며 읽습니다.

5형식 핵심

S + V + O + OC = 주어가 / 목적어를 / 목적격보어 상태로 만들거나, 보거나, 부르거나, 유지한다.
OC 형태대표 구조뜻
형용사make / keep / leave + O + adjectiveO를 어떤 상태로 만들다/유지하다/남겨두다
명사call / name / elect + O + nounO를 어떤 사람/역할/이름으로 부르다
전치사구leave / keep / put + O + prepositional phraseO를 어떤 위치/상태에 두다
to부정사want / need / ask / tell + O + to VO가 어떤 행동을 하기를 원하다/요구하다
원형동사make / let / have / see / hear + O + VO가 어떤 행동을 하게 하다/하는 것을 보다·듣다
현재분사/과거분사see + O + -ing, get + O + p.p.O가 하고 있는 중/된 상태를 말하다

1. 목적격보어가 형용사인 5형식

형용사는 목적어의 상태를 설명합니다. “O를 OC하게 만들다/유지하다/느끼다”로 풀이합니다.

2. 목적격보어가 명사인 5형식

명사는 목적어의 이름, 역할, 신분, 평가를 다시 설명합니다.

3. 목적격보어가 전치사구인 5형식

전치사구는 목적어의 위치나 상태를 설명합니다.

4. 목적격보어가 to부정사/원형동사/-ing인 5형식

want/need는 to V, let/make/have/지각동사는 원형동사, 진행 중인 장면은 -ing가 자주 옵니다.

5. 목적격보어가 p.p.인 5형식

목적어가 완료/처리/수리/준비된 상태가 되게 하는 구조입니다.
